How to start Charge Point Operator and how to connecto Charge Point:

Run the main.py file in the cpo_main folder.

Then you can access fastapi interface from your browser by typing 
                    127.0.0.1:8000/api/16/docs
in the search bar of a web browser to access the docs.

Once the CPO is operational you run the charge_point.py code to connect the simulated charge point.
